# Scientific Computing and Numerical Methods using Python

This repository contains Python implementations of important concepts from **Scientific Computing and Numerical Linear Algebra**.  
The project demonstrates matrix operations, vector norms, numerical methods for solving linear systems, and matrix decompositions.

---

## Topics Covered

### 1. Linear Equation
Implementation of a simple linear equation:

y = mx + c

Python function to compute the value of y.

---

### 2. Matrix Operations
Using NumPy for basic matrix operations:

- Matrix Addition
- Matrix Subtraction
- Matrix Multiplication
- Determinant
- Inverse
- Eigenvalues
- Eigenvectors

---

### 3. Vector and Matrix Norms

#### Vector Norm
- Euclidean Norm

#### Matrix Norms
- Frobenius Norm
- 1-Norm (Maximum Column Sum)
- Infinity Norm (Maximum Row Sum)
- Spectral Norm

---

### 4. Numerical Methods for Linear Systems

Implemented iterative and direct methods:

- Jacobi Method
- Gauss-Seidel Method
- Gauss Elimination Method

The program also shows **convergence comparison using graphs**.

---

### 5. Matrix Decomposition Methods

Implemented important matrix factorization techniques:

- LU Decomposition
- QR Decomposition (Gram-Schmidt)
- Singular Value Decomposition (SVD)

---
